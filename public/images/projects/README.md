# Project Images

Place your project screenshots and images in this folder.

## Required Images:

### For Project Cards (Thumbnails):
- `chatbot-thumb.jpg` - AI-Powered Chatbot Platform thumbnail
- `analytics-thumb.jpg` - ML Analytics Dashboard thumbnail
- `vision-thumb.jpg` - Computer Vision Application thumbnail
- `careerconnect-thumb.jpg` - CareerConnect platform thumbnail

**Recommended size:** 800x600px or 16:9 aspect ratio

### For Project Details (Gallery):
Place additional screenshots in numbered format. **You can add up to 8 images per project!**

The gallery shows images **stacked vertically** in a beautiful scrollable modal.

#### Chatbot Project (up to 6 images):
- `chatbot-1.jpg` through `chatbot-6.jpg`

#### Analytics Project (up to 6 images):
- `analytics-1.jpg` through `analytics-6.jpg`

#### Computer Vision Project (up to 6 images):
- `vision-1.jpg` through `vision-6.jpg`

#### CareerConnect Project (up to 8 images):
- `careerconnect-1.jpg` through `careerconnect-8.jpg`

**Recommended size for gallery:** 1920x1080px or larger (Full HD)

**New Feature:** Images are displayed **full-width** in a vertical scrollable gallery with beautiful animations!

## Image Format Guidelines:
- Use JPG or PNG format
- Optimize images for web (compress before uploading)
- Use descriptive filenames
- Keep file sizes under 500KB for thumbnails, under 2MB for gallery images

## Quick Start:
1. Take screenshots of your projects
2. Resize and optimize them
3. Place them in this folder with the naming convention above
4. The portfolio will automatically display them
